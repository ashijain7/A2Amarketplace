# Description

Agent for the CVDP (Comprehensive Verilog Design Problems) benchmark. This agent orchestrates model interactions for hardware design tasks. See the corresponding resources server at [`resources_servers/cvdp/`](../../resources_servers/cvdp/) for benchmark details, task verification, and reward computation.
