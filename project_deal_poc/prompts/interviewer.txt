You are designing a tiny simulated marketplace, modelled on Anthropic's
"Project Deal" experiment. Your job is to invent {n} fictional people
who will be represented by AI agents negotiating with each other.

Each person has:
- a name (one word, easy to read)
- 1-2 items they want to SELL, each with a floor_price (the minimum
  they will accept)
- 1-2 items they want to BUY, each described loosely and with a
  ceiling_price (the maximum they will pay)
- a short "style" note describing how their agent should negotiate
  (friendly, firm, playful, terse, etc.)

CRITICAL REQUIREMENTS for the group as a whole:
1. There must be MEANINGFUL OVERLAP between sellers and buyers. At
   least 3-4 plausible deals should be possible across the group. If
   Person A wants to sell a "vintage camera", at least one other
   person should want to buy a camera.
2. Items should be everyday things people would actually trade:
   used electronics, books, kitchen gadgets, sports equipment, small
   collectibles, services like "two hours of dog-sitting", etc. No
   houses, cars, or anything over $100.
3. Prices should be realistic and create room for negotiation. If a
   seller's floor is $25, a plausible buyer's ceiling should be $30-50.
   Don't make every deal trivially easy or trivially impossible.
4. Vary the negotiation styles. Don't make everyone "friendly". Mix
   in firm, playful, terse, philosophical, or themed personalities
   (one cowboy is fine, more than one is too much).
5. Floor and ceiling prices should be integers in USD.

Return ONLY a valid JSON array. No prose before or after. No markdown
fences. The schema is:

[
  {{
    "name": "Ravi",
    "items_to_sell": [
      {{
        "item_id": "kbd_01",
        "name": "Mechanical keyboard, brown switches, used 1yr",
        "floor_price": 25
      }}
    ],
    "items_to_buy": [
      {{
        "want_id": "bag_w1",
        "description": "A decent everyday backpack",
        "ceiling_price": 40
      }}
    ],
    "style": "Friendly but firm. Won't go below floor."
  }}
]

Use snake_case ids like "kbd_01", "bike_01", "book_w1", "bag_w2".
Make ids globally unique across the whole array.
